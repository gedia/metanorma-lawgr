= Metanorma-lawgr: Πρότυπο Ελληνικού Νομοθετήματος

Αυτό το πρότυπο δημιουργεί τη βασική δομή ενός ελληνικού νομοθετικού κειμένου
με τη χρήση του Metanorma-lawgr.

== Δομή αρχείων

[source]
----
document.adoc          ← Κύριο αρχείο (metadata + includes)
metanorma.yml          ← Ρυθμίσεις Metanorma
sections/
  00-preamble.adoc     ← Προοίμιο (Έχοντας υπόψη)
  01-part-a.adoc       ← ΜΕΡΟΣ Α — Γενικές Διατάξεις
  02-part-b.adoc       ← ΜΕΡΟΣ Β — Ουσιαστικές Διατάξεις
  03-part-c.adoc       ← ΜΕΡΟΣ Γ — Μεταβατικές / Τελικές
----

== Μεταγλώττιση

[source,sh]
----
metanorma compile document.adoc -t lawgr -x html
----

== Ιεραρχία δομικών στοιχείων

Η ελληνική νομοθεσία ακολουθεί συγκεκριμένη ιεραρχία:

|===
| Στοιχείο | AsciiDoc | Αρίθμηση

| Βιβλίο     | `[heading=book]`       | ΒΙΒΛΙΟ ΠΡΩΤΟ, ΔΕΥΤΕΡΟ
| Μέρος      | `[heading=part]`       | ΜΕΡΟΣ Α, Β, Γ
| Τμήμα      | `[heading=tmima]`      | ΤΜΗΜΑ Α, Β
| Κεφάλαιο   | `[heading=chapter]`    | ΚΕΦΑΛΑΙΟ Α, Β
| Άρθρο      | `[heading=article]`    | Άρθρο 1, 2, 3
| Υποάρθρο   | `[heading=subarticle]` | 1, 2, 3
| Παράγραφος | `[heading=paragraph]`  | 1, 2, 3
| Custom     | `[heading=custom]`     | Α, Β, Γ
|===

Οι αριθμημένες λίστες (περιπτώσεις) χρησιμοποιούν ελληνικά αριθμητικά: α), β), γ)... στ), ζ)...

== Τύποι εγγράφων (doctype)

- `act` — Νόμος / Κανονισμός (προεπιλογή)
- `pd` — Προεδρικό Διάταγμα
- `ap` — Απόφαση
- `egk` — Εγκύκλιος
- `other` — Άλλο
